📋 表格字段说明
字段名说明必填
title / 标题文档名称必填
url / 链接内网文件或页面地址推荐
category / 分类如：技术文档、市场报告选填
year / 年份如：2024选填
source / 来源如：VNPT、ITU选填
author / 作者文档作者选填
abstract / 摘要内容描述，用于搜索匹配选填
tags / 标签关键词，逗号分隔选填

💡 使用说明
• 更新资料：直接在 Google Sheets 里增删改行，点击「↻」刷新即可同步
• 「资料目录」按分类自动分组展示所有资料，用户无需搜索即可浏览
• 用户点击「打开」会跳转到 URL 字段填写的内网地址
• 建议将此 HTML 上传 Google Drive 并「知道链接的人可查看」分享给团队
